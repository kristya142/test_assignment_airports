1. Папка содержит:
Jupiter notebooks - ноутбуки с заданием, кодом Python и комментариями
Generated data - CSV файлы со сгенерированными данными
Sql reports - SQL файлы с отчетами

2. Как запустить
2.1 Открыть консоль, перейти в папку jupyter notebooks
2.2 Создать и активировать виртуальное окружение Python
python -m venv .venv 
source .venv/bin/activate
2.3 Установить необходимые зависимости из файла requirements.txt (Для работы нам нужны только psycopg2, pandas, sqlalchemy , остальные зависимости - сопутствующие jupyterlab)
pip3 install -r requirements.txt

3. Как загрузить данные в свою базу
В файле Jupiter notebooks/reoprts.ipynb в последней ячейке находится код для заполнения 3-х таблиц в базе.
Для подключения своей базы и заполнения сгенерированными данными, необходимо вставить в ячейку параметры подключения (логин, пароль, порт, название базы)
